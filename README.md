# 🌾 Krishi Sakhi – AI-powered Farming Assistant  

**Krishi Sakhi** is a digital companion designed for Kerala’s smallholder farmers.  
It captures farm details, tracks farming activities, and provides **personalized, timely advisories** in Malayalam (via text + voice).  
The goal is to empower farmers with **localized, AI-driven guidance** that improves productivity and sustainability.  

---

## 🚀 Features  
- 👩‍🌾 **Farmer & Farm Profiling** – capture location, land size, crop, soil type, irrigation method.  
- 🎙️ **Conversational Interface** – interact through **voice and text in Malayalam** using the Web Speech API.  
- 📓 **Activity Tracking** – log farm events like sowing, irrigation, fertilizer use, or pest issues in simple language.  
- 🌦️ **Personalized Advisory** – weather-based suggestions, pest alerts, and context-specific tips.  
- 🔔 **Reminders & Alerts** – get nudges for crop operations, subsidy deadlines, and price trends.  
- 📚 **Knowledge Engine** – integrates local crop calendars, pest data, and best practices for continuous improvement.  

---

## 🛠️ Tech Stack  
- **Frontend:** React + Tailwind CSS  
- **Voice Support:** Web Speech API (Malayalam STT + TTS)  
- **Backend (future scope):** Firebase / Node.js  
- **APIs Integrated:**  
  - OpenWeather (for weather-based advisories)  
  - Bhashini (for Indian language processing)  
  - Agmarknet (for mandi price updates, future scope)
